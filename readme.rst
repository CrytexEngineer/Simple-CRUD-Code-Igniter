Simple Crud Code Igniter
1. Ketikan masuk ke localhost\ci_project\index.php\welcome\show. Jika berhasil anda akan melihat tampilan berikut

2. Untuk menambahkan file, pilih tombol insert di pojok kiri, kemudian anda akan melihat tampilan untuk mengisi data. Selanjutnya isikan data dan pilih submit

3. Selanjutnya , jika berhasil anda akan langsung diarahkan ke halaman berikut, untuk menghapus data pilih “delete” di kolom action
 

4. Anda akan melihat field yang dipilih otomatis terhapus
